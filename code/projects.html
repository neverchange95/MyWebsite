To-Do Applikation für Android Smartphones

Eine Applikation für Android Smartphones, welche in der Programmiersprache Java erstellt wurde. Als Entwicklungsumgebung diente hierbei Android Studio. Die App verwaltet alle Todo´s, die vom Benutzer eingetragen wurden und speichert diese in eine SQL-Datenbank auf meinen Homeserver ab. Dieser läuft auf einen Raspberry Pi 4. Der Sourcecode und eine genauere Beschreibung ist auf meinem Github Profil zu finden:
https://github.com/neverchange95/ToDoApp
